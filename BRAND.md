# Base Base — Brand Guidelines

> **بيس بيس** — أدوات أساسية

## 🎨 Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| `--bb-primary` | `#64748b` | Primary brand color |
| `--bb-primary-light` | `#64748b20` | Backgrounds, hover states |
| `--bb-primary-dark` | `#64748b` | Active states, borders |
| `--bb-secondary` | `#64748b` | Secondary text, dividers |
| `--bb-accent` | `#f59e0b` | Highlights, badges, CTAs |
| `--bb-success` | `#10b981` | Success states |
| `--bb-warning` | `#f59e0b` | Warning states |
| `--bb-danger` | `#ef4444` | Error states |
| `--bb-bg` | `#ffffff` | Page background |
| `--bb-text` | `#1e293b` | Primary text |

## 🔤 Typography

- **Headings:** Inter (EN) / IBM Plex Sans Arabic (AR)
- **Body:** Inter (EN) / IBM Plex Sans Arabic (AR)
- **Code:** JetBrains Mono

## 📐 Logo Usage

| Variant | Size | File | Usage |
|---------|------|------|-------|
| Full logo | 512×512 | `images/base_base-logo.svg` | App screen, about page |
| Favicon | 32×32 | `images/favicon.png` | Browser tab |
| Topbar | 28×28 | `images/base_base-topbar.svg` | Navbar |
| Desktop solid | 54×54 | `icons/desktop_icons/solid/` | Workspace (focused) |
| Desktop subtle | 54×54 | `icons/desktop_icons/subtle/` | Workspace (unfocused) |
| Splash | 256×256 | `images/base_base-splash.svg` | Loading screen |
| Email | 200×60 | `images/base_base-email.png` | Email headers |
| Print | 150×50 | `images/base_base-print.svg` | Print formats |

## 🎯 Icon Style

- **Library:** Tabler Icons v3.30+ via `frappe.visual.icons`
- **Style:** Outlined, 1.5px stroke
- **Sizes:** `sm` (16px), `md` (20px), `lg` (24px), `xl` (32px)

## 📏 Spacing

- Minimum clear space around logo: 1× logo width
- Never stretch, rotate, or recolor the logo
- On dark backgrounds, use the light variant
