## يعمل مع ERPNext الحالي لديك

Base Base لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/base_base
bench --site your-site install-app base_base
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
