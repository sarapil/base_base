# البدء مع Base Base

مرحباً! هذا الدليل يرشدك لإعداد Base Base لأول مرة.

## الخطوة 1: الإعداد الأولي

افتح **إعدادات Base Base** وقم بتهيئة التفضيلات.

## الخطوة 2: إعداد المستخدمين والأدوار

قم بتعيين الأدوار لأعضاء فريقك باستخدام نظام CAPS.

## الخطوة 3: ابدأ الاستخدام

انتقل إلى مساحة العمل الرئيسية وابدأ بإنشاء السجلات.
